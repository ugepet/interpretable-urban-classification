# Data Directory

This directory contains the datasets used in the study.

## Tabular Datasets

- `urban_2000.csv`
- `urban_2005.csv`
- `urban_2010.csv`
- `urban_2015.csv`
- `urban_2020.csv`
- `urban_2024.csv`

These datasets contain the predictor variables used to train and evaluate the machine learning models.

## Predictor Variables

- Blue
- Green
- Red
- NIR
- SWIR
- NDVI
- NDBI
- NightLights
- Population
- Elevation

## Target Variable

- Urban (0 = Non-Urban, 1 = Urban)

## Raster Data

`Lagos_full_stack_100m.tif` contains the stacked predictor layers used for spatial inference.

The band order matches the predictor variables used during model training.
